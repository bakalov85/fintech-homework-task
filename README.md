# Fintech commission task

## Requirements:
- Composer
- PHP >= 8.1
- php-bcmath extension
- php-intl extension

## Installation:
Clone/download this repository locally and run<br>
`composer install`<br>
in the repository's root directory

## To run the script:
Go to console/ directory and run:<br>
`php fees.php input.csv`

## To run tests:
`bin/phpunit tests`
